For setting up the compartment:

(Login as default domain)
Step 1: 
Click hamburger icon and select identity and security

Step 2:
Click compartments and then create compartment.

Step 3:
Give it a name, description and select appropriate parentcomapartment (identity domain) in which you wnat to create the compartment.

Step 4:
Click creat compartment.

Congratulations! You created a compartment.
